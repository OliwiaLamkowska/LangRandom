LANGRANDOM BACKEND (z AI)
==========================

1. Zainstaluj Node.js: https://nodejs.org/
2. Otwórz terminal w folderze langrandom-backend
3. Wpisz: npm install
4. W pliku .env wklej swój klucz API
5. Uruchom serwer: npm start
6. Frontend (strona) może teraz pobierać dane z http://localhost:3000/generate-word
